# FarmsteadCalendar

Мобільний застосунок для планування вирощування рослин на основі календаря сезонних періодів.

---

##  Про проєкт

**FarmsteadCalendar** — це мобільний застосунок для Android, який відображає календар вирощування рослин протягом поточного року. Користувач може обрати конкретний сорт дерева, овоча або квітки, після чого застосунок автоматично відобразить відповідні періоди:

- посадки  
- росту  
- цвітіння  
- удобрення  
- обробки  
- збору врожаю  

Усі етапи візуально позначаються безпосередньо в календарі, що дозволяє зручно планувати сезонні роботи.

---

##  Які проблеми вирішує застосунок?

- **Спрощує пошук інформації**  
  Користувачу не потрібно самостійно шукати строки посадки чи збору врожаю — вся інформація автоматично підтягується з бази даних.

- **Забезпечує легкий доступ**  
  Увесь функціонал застосунку безкоштовний та уся інформація структурована.

- **Відображення процесу вирощування**  
  Періоди відображаються графічно, що значно зручніше за текстові довідники.

- **Оптимізує роботу**  
  Дозволяє заздалегідь планувати навантаження та підготовку ресурсів.

---

##  Основний тип користувача

Застосунок орієнтований на:

> Власників присадибних ділянок, садів або невеликих фермерських господарств, які самостійно займаються вирощуванням рослин та прагнуть систематизувати процес догляду й планування сезонних робіт.

---

##  Технології

- **Мова програмування:** Java  
- **Середовище розробки:** Android Studio  
- **Платформа:** Android  
- **Мінімальна версія Android:** 11 (API 30+)    

---

## Основний функціонал

- Відображення поточного календарного року
- Вибір сорту дерева / овоча / квітки
- Автоматичне завантаження періодів росту з бази даних
- Візуальне позначення агротехнічних етапів у календарі
- Локальне збереження користувацьких виборів

---

##  Аналіз конкурентів

На ринку відсутні мобільні застосунки, які:

- автоматично відображають повний річний цикл розвитку конкретних сортів
- базуються на структурованих даних про сорти
- інтегрують ці дані безпосередньо в календарний формат

Більшість існуючих застосунків зосереджені на:
- плануванні грядок
- веденні журналу садівника
- нагадуваннях про полив
- загальних сезонних порадах

---

##  Приклади схожих застосунків

| Назва | Платформа | Зауваження |
|-------|-----------|------------|
| Planter – Garden Planner | Android | Має планування та строки посадки, але не календар із відображенням періодів росту сорту в році |
| GARDENA smart | Android | Додаток більше про автоматизацію догляду, а не про календарні періоди росту сортів |
| Planta | Android | Фокусується на поливі та догляді домашніх рослин, не на сезонних періодах вирощування |
